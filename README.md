# Barista
Barista is a java version manager, like nvm that uses the openjkd archive as a repository.

## Requirements:
- Debian like Distro or Windows 10+
- Python 3.6+
- Pip3
- Git 

## Getting Started:

### Linux
As root user execute the follow commands: <br>
`git clone https://github.com/Darkkyelfo/barista.git` <br>
`cd barista` <br>
`sudo chmod +x install.sh` <br>
`./install.sh`
